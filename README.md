## 它支持什么版本？
| Version               | Support                                       | 
|-----------------------|-----------------------------------------------| 
| 1.20.1(Forge)         | 活跃                                           | 
| 1.20.1(Fabric)        | 结束支持                                        | 
| 1.19.4(OptiFabric)    | 活跃(长期支持)                                   |

| 1.12.2                | 活跃                                            |
